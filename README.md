# Ayush Sharma - Personal Portfolio Website

A modern, responsive personal portfolio website for Ayush Sharma, showcasing political experience, achievements, and campaign gallery.

## 🌐 Live Demo

[Visit Website](https://ayushsharma.github.io)

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Oswald, Inter)

## 📁 Project Structure
├──index.html # Hindi version (default)
├── index-en.html # English version
├── css/
│ ├── style.css # Main stylesheet
│ └── dark-mode.css # Dark mode overrides
├── js/
│ ├── main.js # Core functionality
│ ├── gallery-carousel.js # Gallery carousel
│ └── sticky-header.js # Header scroll behavior
├── assets/
│ └── images/ # All images and favicons
└── README.md
